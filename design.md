Create a modern, professional, commercial-grade admin panel UI using Streamlit to replace the existing basic UI.

Requirements:

Design & Layout:

* Use a wide layout with a fixed sidebar navigation.
* Sidebar must include:

  * Logo at the top
  * Navigation menu with icons
  * Sections: Dashboard, Settings(edit admin profile, change password, manage users, manage classes, manage students, manage subjects, Logout)
* Main content area should use clean card-based layout.
* Use proper spacing, alignment, and padding for professional appearance.
* Add a top header bar showing page title and user info.

Visual Style:

* Use modern SaaS design principles similar to Stripe, Notion, or Shopify admin panels.
* Implement consistent color palette (primary, secondary, background, accent).
* Use subtle shadows, rounded corners, and card containers.
* Add hover effects and button styling.
* Use modern typography and visual hierarchy.

Components:

* Dashboard page must include:

  * KPI cards (existing features)
  * Recent records table with clean styling (activity logs within app)
* Records page must include:

  * Search bar
  * Filter options
  * Modern styled table/grid
* Settings page must include:

  * Profile section
  * System settings section
  * logo upload section

Technical Requirements:

* Use Streamlit native layout features (columns, containers, tabs, sidebar).
* Use custom CSS injected via st.markdown to enhance styling.
* Ensure responsive layout that works on different screen sizes.
* Use reusable components/functions for cards and layout sections.
* Maintain clean, modular, production-quality code structure.

UX Requirements:

* Clear navigation and visual hierarchy
* Fast and smooth interaction
* Professional spacing and alignment
* No clutter or overlapping elements

Code Quality:

* Write clean, maintainable, modular Python code.
* Follow best practices for Streamlit apps.
* Use functions for reusable UI components.
* Organize code logically.

Goal:
The final result should look like a modern commercial SaaS admin panel, not a basic Streamlit app.
